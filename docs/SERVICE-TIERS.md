## Tier Definitions

### T1 — Awareness
Entry-level GRID assessment. Client receives the full Defensibility Index report across all 7 scoring dimensions. Suitable for firms evaluating their AI governance posture for the first time.

**Deliverables:**
- Defensibility Index score (0–100)
- 7-dimension breakdown with risk tier classification
- Executive summary
- Portal access: Report view only

---

### T2 — Advisory
Full assessment with evidence transparency. Client receives the T1 report plus read access to their immutable evidence ledger, showing the full audit trail of scoring decisions.

**Deliverables:**
- Everything in T1
- Evidence ledger access (read-only)
- Classification dispute rights
- Portal access: Report + ledger

---

### T3 — Full Governance
Comprehensive governance engagement. Client receives the full assessment, evidence ledger, and a tier-scoped closure package including policy framework, implementation roadmap, and ongoing implementation support.

**Deliverables:**
- Everything in T2
- Tier-scoped closure package
- Policy framework document
- Implementation roadmap
- Implementation support engagement
- Governance disclosures
- Portal access: Full
